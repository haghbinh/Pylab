    ویجت‌های پیشرفته در Tkinter: Listbox، Combobox و Scrollbar

    در برنامه‌های GUI بزرگ، گاهی نیاز به انتخاب از لیست طولانی، یا پیمایش محتوا داریم. Tkinter ویجت‌های پیشرفته‌ای برای این موارد دارد:

    • Listbox: نمایش لیستی از آیتم‌ها و انتخاب یک یا چند مورد.
    • Combobox: یک منوی کشویی برای انتخاب یک گزینه.
    • Scrollbar: پیمایشگر برای Listbox یا Text.

    ۱. Listbox و Scrollbar 

    import tkinter as tk

root = tk.Tk()
root.title("Listbox با Scrollbar")
root.geometry("300x250")

# ایجاد Scrollbar
scrollbar = tk.Scrollbar(root)
scrollbar.pack(side="right", fill="y")

# ایجاد Listbox
listbox = tk.Listbox(root, yscrollcommand=scrollbar.set)
for item in range(1, 21):
    listbox.insert("end", f"آیتم {item}")
listbox.pack(pady=20)

# اتصال Scrollbar به Listbox
scrollbar.config(command=listbox.yview)

root.mainloop()

    در این مثال، با Scrollbar می‌توان لیست طولانی را پیمایش کرد و انتخاب آیتم‌ها با Listbox انجام می‌شود.

    ۲. Combobox 

    import tkinter as tk
from tkinter import ttk

root = tk.Tk()
root.title("Combobox مثال")
root.geometry("300x150")

def show_selection(event):
    label_result.config(text=f"انتخاب شما: {combo.get()}")

# Combobox
combo = ttk.Combobox(root, values=["گزینه ۱", "گزینه ۲", "گزینه ۳"])
combo.pack(pady=20)
combo.bind("<<ComboboxSelected>>", show_selection)

label_result = tk.Label(root, text="")
label_result.pack()

root.mainloop()

    Combobox برای انتخاب از یک منوی کشویی مناسب است و با رویداد <<ComboboxSelected>> می‌توان انتخاب کاربر را مدیریت کرد.

    ۳. ترکیب ویجت‌ها برای یک فرم پیشرفته 

    import tkinter as tk
from tkinter import ttk

root = tk.Tk()
root.title("فرم پیشرفته")
root.geometry("400x300")

tk.Label(root, text="انتخاب محصول:").pack(pady=5)
combo_product = ttk.Combobox(root, values=["Laptop", "Phone", "Tablet"])
combo_product.pack(pady=5)

tk.Label(root, text="لیست موجودی:").pack(pady=5)
scrollbar = tk.Scrollbar(root)
scrollbar.pack(side="right", fill="y")

listbox_inventory = tk.Listbox(root, yscrollcommand=scrollbar.set)
for i in range(1, 31):
    listbox_inventory.insert("end", f"محصول شماره {i}")
listbox_inventory.pack(pady=5)

scrollbar.config(command=listbox_inventory.yview)

def show_selection():
    selected = combo_product.get()
    label_result.config(text=f"محصول انتخابی: {selected}")

tk.Button(root, text="نمایش انتخاب", command=show_selection).pack(pady=10)
label_result = tk.Label(root, text="")
label_result.pack()

root.mainloop()

    تمرین

    ۱. یک پنجره بسازید و Listbox با حداقل ۱۵ آیتم اضافه کنید.

    ۲. یک Scrollbar به Listbox متصل کنید تا لیست قابل پیمایش باشد.

    ۳. یک Combobox با ۵ گزینه ایجاد کنید و انتخاب آن را در یک Label نمایش دهید.

    ۴. یک دکمه بسازید که آیتم انتخاب شده از Combobox را به Listbox اضافه کند.

    ۵. ترکیبی از Listbox، Combobox و Scrollbar بسازید تا یک فرم تعاملی کامل ایجاد شود.
