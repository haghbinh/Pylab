    توابع پیشرفته در پایتون (Functional Programming)

    در برنامه‌نویسی تابعی (Functional Programming) توابع به عنوان موجودیت‌های درجه یک (first-class) در نظر گرفته می‌شوند.
    یعنی می‌توان توابع را به متغیرها نسبت داد، به توابع دیگر ارسال کرد و یا از توابع دیگر بازگرداند.
    این سبک برنامه‌نویسی به شما کمک می‌کند تا کدهای کوتاه، واضح و قدرتمند بنویسید.

    ۱. توابع Lambda

    توابع Lambda توابع کوچک و ناشناس هستند که می‌توانند چندین ورودی داشته باشند ولی تنها یک خروجی تولید می‌کنند.
    از Lambda معمولاً در مواقعی استفاده می‌کنیم که نیاز به یک تابع کوتاه داریم. 

    def add(x, y):
    return x + y

print(add(2, 3))   
    5
    # Same with lambda
add_lambda = lambda x, y: x + y
print(add_lambda(2, 3))   
    5
    # Lambda with one argument
square = lambda x: x**2
print(square(6))  
    36
    # Lambda with no arguments
greet = lambda: "Hello!"
print(greet()) 
    Hello!

    ۲. تابع map()

    تابع map یک تابع را روی همه عناصر یک Iterable مانند لیست یا Tuple اعمال می‌کند و نتیجه را بازمی‌گرداند. 

    numbers = [1, 2, 3, 4, 5]

# Square each number using map and lambda
squares = list(map(lambda x: x**2, numbers))
print(squares)   
    [1, 4, 9, 16, 25]
    # Convert list of strings to integers
str_numbers = ["1", "2", "3"]
int_numbers = list(map(int, str_numbers))
print(int_numbers) 
    [1, 2, 3]
    # Add corresponding elements of two lists
a = [1, 2, 3]
b = [4, 5, 6]
summed = list(map(lambda x, y: x + y, a, b))
print(summed)  
    [5, 7, 9]

    ۳. تابع filter()

    تابع filter عناصر یک Iterable را بر اساس یک شرط (تابعی که True یا False برمی‌گرداند) فیلتر می‌کند. 

    numbers = [10, 15, 20, 25, 30]

# Get even numbers using filter and lambda
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)   # Output: [10, 20, 30]
    [10, 20, 30]
    # Filter numbers greater than 18
greater_than_18 = list(filter(lambda x: x > 18, numbers))
print(greater_than_18)  # Output: [20, 25, 30]
    [20, 25, 30]
    # Filter strings starting with 'a'
words = ["apple", "banana", "avocado", "cherry"]
a_words = list(filter(lambda w: w[0] == 'a', words))
print(a_words)  # Output: ['apple', 'avocado']
    ['apple', 'avocado']

    ۴. تابع zip()

    تابع zip چند Iterable را با هم ترکیب می‌کند و یک Iterable از Tupleها ایجاد می‌کند. 

    names = ["Alice", "Bob", "Charlie"]
scores = [85, 90, 78]
ages = [20, 21, 19]

# Combine two lists
result = list(zip(names, scores))
print(result)  
    [('Alice', 85), ('Bob', 90), ('Charlie', 78)]
    # Combine three lists
result3 = list(zip(names, scores, ages))
print(result3)
    [('Alice', 85, 20), ('Bob', 90, 21), ('Charlie', 78, 19)]
    # Unzip a list of tuples
zipped = [('Alice', 85), ('Bob', 90), ('Charlie', 78)]
names2, scores2 = zip(*zipped)
print(names2)   
    ('Alice', 'Bob', 'Charlie')
    print(scores2)
    (85, 90, 78)

    ۵. ترکیب map، filter و zip

    می‌توان این توابع را با هم ترکیب کرد تا کدهای کوتاه و خوانا نوشت. 

    numbers = [1, 2, 3, 4, 5, 6]

# Filter even numbers
evens = list(filter(lambda x: x % 2 == 0, numbers))

# Square the filtered numbers
squared_evens = list(map(lambda x: x**2, evens))
print(squared_evens)   
    [4, 16, 36]
    # Combine two lists, filter and square
a = [1, 2, 3, 4]
b = [5, 6, 7, 8]
result = list(map(lambda x: x**2, filter(lambda x: x % 2 == 0, a)))
print(result)  
    [4, 16]

    تمرین

    ۱. یک Lambda بنویسید که دو عدد را با هم جمع کند و نتیجه را چاپ کند.

    ۲. لیستی از اعداد بسازید و با map هر عدد را سه برابر کنید.

    ۳. لیستی از اعداد بسازید و با filter فقط اعداد فرد را انتخاب کنید.

    ۴. دو لیست بسازید و با zip آن‌ها را به یک لیست از Tupleها تبدیل کنید.

    ۵. با ترکیب map و filter لیستی از اعداد ایجاد کنید و ابتدا اعداد بزرگتر از ۵ را فیلتر کنید و سپس هر عدد را به توان دو برسانید.

    ۶. یک لیست از رشته‌ها بسازید و با filter فقط رشته‌هایی که با حرف ‘a’ شروع می‌شوند انتخاب کنید.

    ۷. دو لیست بسازید و با map مجموع عناصر متناظر آن‌ها را محاسبه کنید.

    ۸. یک لیست از Tuple بسازید و با unzip (استفاده از *) دوباره به دو لیست جداگانه تبدیل کنید.
