    Iterables و Iterators در پایتون

    در پایتون، بسیاری از اشیاء قابل تکرار هستند؛ مانند لیست‌ها، تاپل‌ها، رشته‌ها و دیکشنری‌ها.
    به این اشیاء Iterable می‌گوییم، زیرا می‌توانیم آن‌ها را تکرار کنیم (با حلقه for یا سایر توابع).
    Iterator ابزاری است که روی یک Iterable حرکت می‌کند و عنصر بعدی را برمی‌گرداند.

    ۱. Iterable چیست؟

    یک شئ Iterable باید متد iter() داشته باشد. لیست‌ها، تاپل‌ها، رشته‌ها و دیکشنری‌ها نمونه‌هایی از Iterable هستند. 

    # List is an iterable
numbers = [1, 2, 3, 4]
for num in numbers:
    print(num)

# String is also iterable
text = "Python"
for char in text:
    print(char)

    ۲. Iterator چیست؟

    Iterator یک شئ است که متد next() دارد و با آن می‌توان عناصر یک Iterable را یکی‌یکی دریافت کرد. برای دریافت Iterator از یک Iterable، از تابع iter() استفاده می‌کنیم. 

    numbers = [10, 20, 30]
it = iter(numbers)

print(next(it))  
    10
    print(next(it))  
    20
    print(next(it))  
    30
    print(next(it))

    ۳. حلقه for و Iterators

    حلقه for به صورت خودکار از Iterable یک Iterator می‌سازد و با next() عناصر را دریافت می‌کند. بنابراین کد کوتاه و خوانا می‌شود. 

    numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num)
    1
2
3
4
5

    ۴. ساخت Iterator اختصاصی

    می‌توان کلاس‌هایی ساخت که خودشان Iterator باشند. برای این کار باید متدهای iter() و next() را تعریف کنیم. 

    class MyRange:
    def __init__(self, start, end):
        self.current = start
        self.end = end
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.current >= self.end:
            raise StopIteration
        value = self.current
        self.current += 1
        return value

for i in MyRange(5, 10):
    print(i)
    5
6
7
8
9

    ۵. توابع کمکی با Iterators

    پایتون توابعی مانند enumerate()، zip() و reversed() تولید Iterator می‌کنند و می‌توان از آن‌ها در حلقه for استفاده کرد. 

    # enumerate example
words = ["apple", "banana", "cherry"]
for idx, word in enumerate(words):
    print(idx, word)

# zip returns iterator
numbers = [1, 2, 3]
letters = ['a', 'b', 'c']
for n, l in zip(numbers, letters):
    print(n, l)
    0 apple
1 banana
2 cherry
1 a
2 b
3 c
    # reversed returns iterator
for n in reversed(numbers):
    print(n)
    3
2
1

    تمرین

    ۱. یک لیست بسازید و با حلقه for تمام عناصر آن را چاپ کنید.

    ۲. یک Iterator از یک رشته بسازید و با next() عناصر آن را یکی‌یکی چاپ کنید.

    ۳. یک کلاس بسازید که یک محدوده عددی را به صورت Iterator برگرداند.

    ۴. با استفاده از enumerate، ایندکس و مقدار عناصر یک لیست را چاپ کنید.

    ۵. دو لیست بسازید و با zip آن‌ها را ترکیب کرده و عناصر آن را چاپ کنید.

    ۶. یک Iterator بسازید و در یک حلقه while با next() تمام عناصر آن را چاپ کنید تا StopIteration رخ دهد.
