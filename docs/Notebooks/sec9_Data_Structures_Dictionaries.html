    ساختار داده‌ها در پایتون: دیکشنری‌ها (Dictionaries)

    دیکشنری‌ها یکی از پرکاربردترین ساختارهای داده‌ای در پایتون هستند.
    هر دیکشنری شامل کلید (Key) و مقدار (Value) است.
    ویژگی مهم دیکشنری‌ها این است که کلیدها یکتا هستند و به وسیله کلیدها به مقادیر دسترسی پیدا می‌کنیم.

    ۱. ایجاد دیکشنری

    برای تعریف دیکشنری از آکولاد {} استفاده می‌کنیم و هر کلید با مقدار مربوطه با دو نقطه : جدا می‌شود. 

    # Creating dictionaries
student = {"name": "Ali", "age": 20, "major": "Physics"}
grades = {"math": 18, "physics": 19, "chemistry": 17}

print(student)
print(grades)
    {'name': 'Ali', 'age': 20, 'major': 'Physics'}
{'math': 18, 'physics': 19, 'chemistry': 17}

    ۲. دسترسی به مقادیر

    برای دسترسی به مقدار مربوط به یک کلید از [key] یا متد get() استفاده می‌کنیم. 

    student = {"name": "Ali", "age": 20, "major": "Physics"}

print(student["name"])      # Ali
print(student.get("age"))   # 20
    Ali
20

    ۳. اضافه کردن و تغییر مقادیر

    می‌توانیم یک کلید جدید اضافه کنیم یا مقدار یک کلید موجود را تغییر دهیم. 

    student = {"name": "Ali", "age": 20}

# Adding a new key
student["major"] = "Physics"
print(student)

# Modifying an existing key
student["age"] = 21
print(student)
    {'name': 'Ali', 'age': 20, 'major': 'Physics'}
{'name': 'Ali', 'age': 21, 'major': 'Physics'}

    ۴. حذف عناصر

    برای حذف عناصر از del یا pop استفاده می‌کنیم. 

    student = {"name": "Ali", "age": 21, "major": "Physics"}

# Remove by key
del student["major"]
print(student)

# Remove and return value
age = student.pop("age")
print(student)
print("Removed age:", age)
    {'name': 'Ali', 'age': 21}
{'name': 'Ali'}
Removed age: 21

    ۵. حلقه روی دیکشنری

    می‌توانیم روی کلیدها، مقادیر یا هر دو با هم حلقه بزنیم. 

    grades = {"math": 18, "physics": 19, "chemistry": 17}

# Loop over keys
for subject in grades:
    print(subject)

# Loop over values
for score in grades.values():
    print(score)

# Loop over key-value pairs
for subject, score in grades.items():
    print(subject, "->", score)
    math
physics
chemistry
18
19
17
math -> 18
physics -> 19
chemistry -> 17

    ۶. توابع و متدهای مفید

    دیکشنری‌ها متدهای متعددی دارند که بسیار مفید هستند. 

    student = {"name": "Ali", "age": 21, "major": "Physics"}

print(student.keys())    # List of keys
print(student.values())  # List of values
print(student.items())   # List of (key, value) tuples
print(len(student))      # Number of items
    dict_keys(['name', 'age', 'major'])
dict_values(['Ali', 21, 'Physics'])
dict_items([('name', 'Ali'), ('age', 21), ('major', 'Physics')])
3

    تمرین

    ۱. یک دیکشنری شامل نام و سن چند نفر بسازید و تعداد افراد را چاپ کنید.

    ۲. دیکشنری‌ای بسازید و یک کلید جدید با مقدار دلخواه اضافه کنید.

    ۳. دیکشنری‌ای بسازید و یک کلید مشخص را حذف کنید و مقدار آن را چاپ کنید.

    ۴. دیکشنری‌ای بسازید و با استفاده از حلقه، کلیدها و مقادیر را جداگانه چاپ کنید.

    ۵. یک دیکشنری بسازید و با استفاده از متدهای keys، values و items آن را بررسی کنید.
