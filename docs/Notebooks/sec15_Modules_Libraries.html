    ماژول‌ها و کتابخانه‌ها در پایتون (Modules & Libraries)

    ماژول‌ها فایل‌هایی شامل کد پایتون هستند که می‌توان آن‌ها را وارد (import) کرد تا از توابع، کلاس‌ها و متغیرهای آن‌ها استفاده کرد.
    کتابخانه‌ها مجموعه‌ای از ماژول‌ها هستند که قابلیت‌های بیشتری را در اختیار برنامه‌نویس قرار می‌دهند.
    استفاده از ماژول‌ها و کتابخانه‌ها باعث می‌شود کد شما سازمان‌دهی شده، قابل استفاده مجدد و کوتاه‌تر باشد.

    ۱. وارد کردن ماژول‌ها (import)

    برای استفاده از ماژول‌ها از دستور import استفاده می‌کنیم. 

    # Import entire module
import math

print(math.sqrt(16))  
    4.0
    print(math.pi)      
    3.141592653589793
    # Import specific function
from math import factorial
print(factorial(5))   
    120
    # Import with alias
import math as m
print(m.sin(m.pi/2))  
    1.0

    ۲. ماژول‌های استاندارد پایتون

    پایتون دارای ماژول‌های استاندارد فراوانی است که نیاز به نصب جداگانه ندارند. چند مثال معروف: 

    import random

# Generate random number between 1 and 10
print(random.randint(1, 10))
    3
    # Shuffle a list
numbers = [1, 2, 3, 4, 5]
random.shuffle(numbers)
print(numbers)
    [5, 3, 1, 2, 4]
    import datetime

# Get current date and time
now = datetime.datetime.now()
print(now)
    2025-08-25 14:47:32.849006

    ۳. نصب و استفاده از کتابخانه‌های خارجی

    می‌توان کتابخانه‌های خارجی را با ابزار pip نصب کرد و سپس استفاده نمود. 

    # Example: installing requests library
# pip install requests

import requests

response = requests.get("https://api.github.com")
print(response.status_code)  
    200
    print(response.headers['Content-Type'])
    application/json; charset=utf-8

    ۴. ایجاد ماژول اختصاصی

    می‌توان خودمان فایل پایتون بسازیم و آن را به عنوان ماژول وارد برنامه کنیم. 

    # File: mymodule.py
def greet(name):
    return f"Hello, {name}!"

def add(x, y):
    return x + y

# Main program
import mymodule

print(mymodule.greet("Ali"))  # Output: Hello, Ali!
print(mymodule.add(5, 7))     # Output: 12

    تمرین

    ۱. از ماژول math تابع pow را برای محاسبه توان یک عدد استفاده کنید.

    ۲. لیستی از اعداد بسازید و با ماژول random یک عدد تصادفی از آن انتخاب کنید.

    ۳. با استفاده از ماژول datetime تاریخ و زمان فعلی را چاپ کنید.

    ۴. یک فایل پایتون بسازید و توابع دلخواه ایجاد کنید، سپس آن را به عنوان ماژول وارد برنامه اصلی کنید.

    ۵. یک کتابخانه خارجی مثل requests یا numpy نصب کنید و از یک قابلیت آن استفاده کنید.

    ۶. با import as یک ماژول را با نام مستعار وارد کنید و یک تابع آن را فراخوانی کنید.
